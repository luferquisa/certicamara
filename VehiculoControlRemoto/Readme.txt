Para ejecutar el programa se puede con una de las dos siguientes formas.

Forma 1:

1. Abriendo el proyecto en Netbeans
2. Corriendo la clase VehiculoControlRemoto.java

Forma 2

1. En un terminal o CMD dirigirse a la carpeta donde se descargo el proecto
	cd ruta\descarga\proyecto
2. Una vez ahi ir a la carpeta dis
	cd dist
3. Una vez ahi ejecutar 
	java -jar VehiculoControlRemoto.jar
	
Uso del programa	
	1. Primero solicita las dimensiones del tablero
	2. A continuación el programa dibuja el tablero y la posicion inicial del carro
	3. Luego se pueden digitar las coordenas para el movimiento en formato individual (x,D) donde x es el número de pasos y D la dirección, o varios comandos 
	   separados por ;
	4. Para terminar el programa se digita un caracter c
